## 第一部分：使用门槛与可用性（中国大陆用户视角）

本节逐一说明每个服务对大陆用户的可用性，重点区分"能否注册/登录"与"是否需要开户/交易"。

### 1.2 监管背景（关键上下文）

2025年，中国证监会等八部门联合印发《综合整治非法跨境证券期货基金经营活动实施方案》，对境外互联网券商展开整治：

- **2025年5月22日**：证监会宣布依法对老虎证券、富途证券、长桥证券三家境外互联网券商展开调查，并拟没收其境内外全部非法所得。
- **2025年9月22日**：富途证券、老虎证券进一步关闭了中国内地居民的开户通道。根据最新监管要求，目前中国内地客户开户需持有**海外永居身份或港澳身份证**。
- **2025年9月25日**：富途牛牛客服强调，现阶段仅支持有中国香港或澳门身份证的客户开户；待系统升级完成后，客户可使用**内地身份证 + 海外永居身份证明**进行开户。

**受影响的平台**：富途（Futu/Moomoo）、老虎证券（Tiger Brokers）、长桥证券（Longbridge）。

**不受影响的平台**：国信证券、广发证券（均为中国大陆持牌券商）。

### 1.3 各平台详细说明

**✅ 对大陆用户完全可用**

**国信证券 Skills（小信智慧助手）**
- 注册/登录：仅需手机号验证码登录，无需开通交易账户
- 使用条件：登录后即可使用所有已上线 Skill，无资金门槛
- 费用：限时免费
- 大陆用户可用性：✅ 完全可用

**广发证券 MCP（gfconnect/mcp）**
- 注册/登录：需广发通账号，完成身份认证即可，无需开通交易账户
- 使用条件：获取 Bearer Token 后接入，无资金门槛
- 费用：免费
- 大陆用户可用性：✅ 完全可用

---

**❌ 对大陆用户实质上不可用（新用户）**

**Moomoo Skills（富途海外版）**

- 存量用户：已有账户的用户可能仍可继续使用
- 大陆用户可用性：❌ 新用户不可用；⚠️ 存量用户可能可用

**老虎证券 Skills（tigeropen-skill）**
- 注册/登录：2025年9月后，大陆用户新开户需提供海外永居身份或港澳身份证
- 使用条件：需老虎证券账户 + API 权限
- 存量用户：已有账户的用户可能仍可继续使用
- 大陆用户可用性：❌ 新用户不可用；⚠️ 存量用户可能可用

**长桥 Longbridge Skills**
- 注册/登录：2025年9月后，大陆用户新开户需提供海外永居身份或港澳身份证
- 使用条件：需 Longbridge 账户 + OpenAPI 权限
- 存量用户：已有账户的用户可能仍可继续使用
- 大陆用户可用性：❌ 新用户不可用；⚠️ 存量用户可能可用

### 2.2 场景一：宏观经济研究

**当您需要** GDP、CPI、PPI、货币供应、利率、就业、进出口、大宗商品价格等宏观数据，用于宏观策略研究或市场观察时——

**唯一选择：国信证券 Skills（宏观经济数据）**

其他平台（广发、富途、老虎、长桥、Moomoo）均未提供专门的宏观经济数据 Skill。

**国信证券「宏观经济数据」能拿到什么：**

- **核心指标**：GDP（实际/名义/平减指数/分产业）、CPI（同比/环比/核心/食品/能源）、PPI、货币供应量、利率、就业数据、进出口数据
- **商品价格**：原油、黄金、有色金属等具体品种价格与库存数据
- **查询维度**：支持时间维度（近三年、近十年）与地域维度（国家、区域、省份）灵活组合
- **输出方式**：自动生成本地文件，并附带指标说明
- **数据来源**：国信证券专业宏观数据库
- **覆盖经济体**：全球主要经济体（中国、美国、欧盟等）

### 2.3 场景二：行情数据分析

**当您需要** A股/港股的实时行情、历史 K 线、资金流向、板块与概念分析，用于盯盘或交易决策时——

**首选：国信证券 Skills（行情数据）**

**国信证券「行情数据」能拿到什么：**

- **实时行情**：最新价、开盘价、最高价、最低价、昨收价、涨跌幅、成交量、成交额、换手率、市盈率、市净率（A股、港股）
- **资金流向**：主力资金净流入/流出金额、超大单/大单/中单/小单资金流向、北向资金/南向资金流向、机构持仓变化（个股、板块维度）
- **板块分析**：行业板块指数、概念板块指数、板块涨跌幅、板块成交额、板块资金净流入/流出、板块内个股涨跌分布
- **历史数据**：日 K、周 K、月 K、分时数据、逐笔成交
- **覆盖范围**：A股（沪深两市）、港股、主要指数

**其他平台情况：**
- 广发证券 MCP：lh b/quant/windmill 均不直接提供日常实时行情查询能力
- 富途/老虎/长桥/Moomoo：行情数据能力强，但大陆用户新注册已受限

### 2.4 场景三：上市公司财务深度分析

**当您需要** 获取上市公司财务报表、计算财务指标、进行同行业对比，用于基本面投资研究时——

**两个选择均可：国信证券 Skills（财务数据）或广发证券 MCP（quant）**

两者侧重点不同，可以同时使用。

**国信证券「财务数据」能拿到什么：**

- **财务三表查询**：利润表、资产负债表、现金流量表核心数据获取
- **核心指标体系**：覆盖盈利能力、成长能力、偿债能力、运营能力等 **300+ 财务指标**
- **多维度数据支持**：支持年度、季度及 TTM 数据查询
- **多公司对比分析**：支持同指标下多标的横向对比
- **结构化结果输出**：自动生成本地文件，并附带数据说明
- **覆盖范围**：A股、港股
- **数据来源**：国信证券财务数据库，覆盖沪深及港股上市公司财务数据，按照披露规则更新

**广发证券 MCP「quant（财务分析）」能拿到什么：**

- **5 大分析维度**：估值分析、盈利能力、现金流分析、偿债能力、成长性分析
- **17 个独立工具**：结构化程度高，便于 AI Agent 精确调用
- **覆盖市场**：沪深 A股上市公司
- **适用场景**：上市公司财务数据对比分析（估值/盈利/现金流/偿债/成长性）

**对比建议：**
- 如果您需要**指标数量多、时间维度灵活**（年度/季度/TTM）、**多公司横向对比**：选国信证券
- 如果您需要**结构化工具调用**、**与龙虎榜/指数估值分析联动使用**：选广发证券（同一 MCP 服务内可同时调用多个工具）

### 2.5 场景四：龙虎榜分析（短线/题材炒作研究）

**当您需要** 分析沪深两市触发龙虎榜规则的个股、查询上榜营业部的交易明细、研究短线资金动向时——

**唯一选择：广发证券 MCP（lh b 服务）**

国信证券无龙虎榜相关功能。

**广发证券「龙虎榜分析（lh b）」能拿到什么：**

共提供 **9 个工具**：

| 工具名称 | 分析对象 | 可获取数据 |
|---------|---------|---------|
| `lhb_aborttrade_market_date_get` | 指定日期、指定市场的龙虎榜个股 | 个股代码、个股名称、上榜原因、上榜日期 |
| `lhb_aborttrade_market_code_date_get` | 指定日期、指定市场的单只个股 | 个股龙虎榜明细：买卖营业部、买卖金额、席位性质（机构/营业部） |
| `lhb_aborttrade_stock_market_code_get` | 指定市场的单只个股 | 个股上榜历史列表：上榜日期、上榜原因、累计上榜次数 |
| `lhb_stat_stock_months_get` | 指定时间区间内的全市场上榜个股 | 上榜个股排行：个股代码、上榜次数、累计买入金额、累计卖出金额 |
| `lhb_stat_stock_market_code_months_get` | 指定时间区间内的单只个股 | 个股龙虎榜统计数据：上榜次数、买卖总额、资金净流入/流出 |
| `lhb_stat_dept_id_months_get` | 指定时间区间内的单个营业部 | 营业部龙虎榜交易统计数据：营业部名称、上榜次数、买卖总额、偏好个股 |
| `lhb_outline_plate_get` | 龙虎榜整体/评级维度 | 龙虎榜概括数据：当日上榜个股数量、总交易金额、市场热度评级 |
| `lhb_calendar_market_month_get` | 指定市场、指定月份的龙虎榜日历 | 龙虎榜日历：有龙虎榜数据的日期列表 |
| `lhb_aborttrade_batch_stock_post` | 指定日期的批量个股龙虎榜情况 | 批量个股龙虎榜情况：多只个股的上榜状态、明细数据 |

**时间维度**：支持近 1 月/3 月/6 月/12 月多时间维度查询（`m1`/`m3`/`m6`/`m12`）。

### 2.6 场景五：指数估值分析 + ETF 联动选品

**当您需要** 判断宽基/行业/主题指数的估值水平（PE/PB 历史分位）、并快速找到跟踪低估指数的 ETF 产品时——

**唯一选择：广发证券 MCP（windmill 服务）**

国信证券无此功能。

**广发证券「指数估值分析（windmill）」能拿到什么：**

- **指数基本信息**：指数代码、指数名称、指数类型（宽基/行业/主题）
- **估值数据**：PE（市盈率）分位值、PB（市净率）分位值、历史估值百分位
- **历史表现**：指数历史涨跌幅、年化收益率、波动率
- **ETF 联动数据**：跟踪该指数的 ETF 列表、ETF 代码、ETF 名称、ETF 规模、ETF 费率
- **投资建议参考**：基于估值分位的投资参考（低估值提示入场机会、高估值提示风险）
- **使用方式**：支持分页查询（`page` + `perPage` 参数）

**实用价值**：当 windmill 显示某宽基指数 PE 分位处于历史 10% 低分位时，可直接查看跟踪该指数的所有 ETF 产品（代码、名称、规模、费率），实现"估值分析 → ETF 选品"一站式完成。

### 2.7 场景六：场外基金多维度对比

**当您需要** 对比多只场外基金的业绩表现、风险指标、持仓结构、基金经理背景，用于基金投资决策时——

**首选：国信证券 Skills（基金对比）**

广发证券无基金对比功能。

**国信证券「基金对比」能拿到什么：**

- **多维度对比分析**：覆盖基金概况、业绩表现、风险指标、资产配置、基金经理等核心维度
- **维度灵活自定义**：支持用户通过设置特定对比维度，按需聚焦核心指标，实现同维度下多只基金精准分析
- **基金标的智能匹配**：根据用户场景化指令，智能匹配对应基金产品
- **筛选结果结构化输出**：支持将对比结果自动生成本地文件，方便后续分析、复盘与标的跟踪
- **适用场景**：多只基金产品横向对比与优选、基金核心指标差异量化分析、不同类型基金优劣综合研判、基金组合构建与标的筛选决策
- **数据来源**：基金公司公告、Wind、恒生聚源等合法公开的信息渠道
- **覆盖范围**：场外基金（开放式基金）

### 2.8 场景七：智能选股

**当您需要** 根据财务指标、技术指标、行情数据、市值规模等多维度条件筛选 A 股个股时——

**首选：国信证券 Skills（智能选股）**

广发证券无选股功能。

**国信证券「智能选股」能拿到什么：**

- **多维度条件自定义筛选**：支持按财务指标、技术指标、行情数据、市值规模等多个维度灵活设置筛选条件
- **多指标组合筛选能力**：支持同时叠加多个筛选条件，对股票进行交叉筛选，提升结果匹配度
- **常见选股维度覆盖**：支持市盈率、市净率、净利润、均线、MACD、KDJ 等常见指标条件设置，便于快速构建筛选规则
- **条件结果结构化输出**：筛选结果以标准化数据形式展示，便于用户查看、分析及后续处理
- **适用场景**：
  - 根据财务指标筛选股票（如市盈率、市净率、净利润等）
  - 根据技术指标选股（如均线、MACD、KDJ 等）
  - 查找满足特定条件的股票组合
  - 行业板块筛选
  - 涨停板、跌停板股票查询
  - 资金流向筛选
- **数据来源**：国信证券专业行情系统

### 2.9 场景八：ETF 产品筛选与多维分析

**当您需要** 从全市场 ETF 产品中筛选符合条件的标的，用于 ETF 投资或资产配置时——

**首选：国信证券 Skills（ETF筛选器）**

广发证券有 etf_rank 服务，但官方文档（etf_rank.md）无法访问，功能详情未知。

**国信证券「ETF筛选器」能拿到什么：**

- **多维度指标自定义筛选**：支持按基本信息、交易属性、基本面、收益表现、风险波动、行情指标等多个维度自定义筛选条件，精准匹配投资需求
- **预设专业方案快捷查询**：内置 T+0 短线突破、热点赛道、高分红低波动、全球市场热门、平衡资产配置等多类专业方案，一键筛选优质 ETF 标的
- **组合方案联动筛选**：支持在专业方案基础上叠加自定义筛选条件，进一步缩小标的范围，兼顾专业筛选逻辑与个性化需求
- **筛选结果结构化输出**：支持将筛选结果自动生成本地文件，方便后续分析、复盘与标的跟踪

### 3.2 各平台使用条件详细说明

**国信证券 Skills**

- **注册/登录要求**：需完成手机号验证码登录，无需开通交易账户
- **开户要求**：无需开户
- **资产/资金门槛**：无
- **费用**：限时免费
- **接入方式**：在 OpenClaw 中复制安装指令，自动完成 Skill 安装
- **运行环境**：需登录获取 API Key
- **适用 AI 工具**：目前主要适配 OpenClaw
- **大陆用户可用性**：✅ 正常，无额外限制

**广发证券 MCP**

- **注册/登录要求**：需广发通账号，完成身份认证即可，无需开通交易账户
- **开户要求**：无需开户
- **资产/资金门槛**：无
- **费用**：免费
- **接入方式**：配置 streamable-HTTP 方式接入，需获取 Bearer Token
- **运行环境**：配置后可在任何支持 MCP 协议的 AI 工具中使用
- **适用 AI 工具**：Claude Code、Cursor、VS Code、OpenClaw 等所有支持 MCP 的工具
- **大陆用户可用性**：✅ 正常，无额外限制

**Moomoo Skills**

- **注册/登录要求**：需注册 Moomoo 账号
- **开户要求**：行情查询、资讯分析类非交易 Skill 仅需注册 Moomoo 账号，无需完成对应市场开户；交易功能需开户
- **资产/资金门槛**：无（但注册本身对大陆新用户已受限）
- **费用**：免费（部分高级行情需订阅）
- **接入方式**：向 AI 工具发送指令自动安装；需打开 OpenD 终端（本地网关）
- **适用 AI 工具**：OpenClaw、Claude Code、Cursor、Codex 等
- **大陆用户可用性**：❌ 2025年9月起新用户注册受限，需海外永居身份或港澳身份证

**富途 Futunn OpenAPI**

- **注册/登录要求**：需富途牛牛账户
- **开户要求**：开通后可免费使用 OpenAPI；交易功能需开户
- **资产/资金门槛**：无（但开户本身对大陆新用户已受限）
- **费用**：开户后免费；大陆用户可免费获取港股 LV2 + A股 LV1 行情（部分高级行情需购买行情卡）
- **接入方式**：参考 Futu API Skill 文档；需下载 Futu OpenD 客户端
- **适用 AI 工具**：OpenClaw、Claude Code、Cursor、Codex 等
- **大陆用户可用性**：❌ 2025年9月起新用户开户受限，需海外永居身份或港澳身份证

**老虎证券 Skills（tigeropen-skill）**

- **注册/登录要求**：需老虎证券账户
- **开户要求**：需开通账户 + API 权限（不确定是否必须开通交易账户）
- **资产/资金门槛**：未明确说明
- **费用**：未明确说明
- **接入方式**：`npx skills add tigerfintech/tigeropen-skill`；需安装 Tiger OpenAPI SDK
- **适用 AI 工具**：Claude Code、Cursor、Gemini CLI、GitHub Copilot、VS Code、Codex、OpenClaw 等

**长桥 Longbridge Skills**

- **注册/登录要求**：需 Longbridge 账户
- **开户要求**：需开通账户 + OpenAPI 权限（不确定是否必须开通交易账户）
- **资产/资金门槛**：未明确说明
- **费用**：接口服务本身免费；部分高级行情需购买
